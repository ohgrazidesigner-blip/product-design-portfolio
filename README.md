# Graziele Costa | Product Design Portfolio

Portfolio website for Graziele Costa, a Product Designer based in Salvador, Brazil. The work focuses on complex workflows, information architecture, UX/UI, and design systems across education, operations, and financial products.

## Selected work

- UniFatecie: independent redesign study for a fragmented student journey
- Optima: independent operational intelligence product concept
- LedgerFlow: independent accounting management product concept

Concept-stage work is identified clearly and is not presented as shipped or validated impact.

## Technology

- React
- TypeScript
- Vite
- Tailwind CSS
- Motion

## Local development

```bash
npm install
npm run dev
```

## Production build

```bash
npm run typecheck
npm run build
npm run preview
```
